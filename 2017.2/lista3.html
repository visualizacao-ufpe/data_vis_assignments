Problema 1

O objetivo deste problema é estudar as ocorrências aeronáuticas que ocorreram na aviação civil brasileira. Para. Para tanto, usaremos este dataset. Sua tarefa é implementar um sistema de visualização que permita mapear as ocorrências dadas no arquivo anterior. Precisamos de três visualizações. A primeira, as ocorrências devem ser mostradas como pontos sobre o mapa de municípios do Brasil (dados). Os pontos devem ser coloridos de acordo com a coluna "ocorrencia_classificacao", vermelho para acidente e azul para incidente. A segunda visualização deve mostrar o número de ocorrências por estado (dados). Escolha uma escala de cores apropriada e mostre uma legenda para as cores escolhidas. Finalmente, a tercerira visualização deve mostrar o número de ocorrências por município brasileiro. Novamente, escolha uma escala de cores apropriada e mostre uma legenda. O usuário deve ser capaz de mudar de uma visualização para outra por meio de um radio button.

Problema 2

Neste problema estudaremos a taxa de óbitos por AIDS nas regiões metropolitanas do Brasil. O conjunto de dados contém dados para cada região metropolitana de 1990 até 2009. Ao invés de explorar estes dados por meio de visualizações de timeseries diretamente, usaremos a abordagem de redução de dimensionalidade para tentar buscar semelhanças entre regiões metropolitanas. Para isso, use o TSNE para projetar estes dados, assumindo que cada ano corresponde a uma dimensão dos dados. Uma vez que os dados estejam projetados, plote os mesmos por meio de círculos, um para cada região metropolitana. Deve ser possível ver o nome de cada região ao se passar o mouse por cima da mesma.

Problema 3

Neste problema estudares o conjunto de artigos de visualização de dados publicados em conferências da área. Sua tarefa é de fazer uma Tag Cloud para cada uma das conferências Vis, InfoVis e VAST dos anos de 2010 até 2016. Você deve exibir 21 tag clouds (uma para combinação conferência ano) usando os abstracts dos papers de cada conferência e ano. Remove stop-words (using this list). Para plotar as tagclouds use a biblioteca d3-cloud.
