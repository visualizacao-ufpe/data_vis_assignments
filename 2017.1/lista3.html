Problema 1

Neste problema usaremos a implementação do algorítimo de Force Directed Layout (FDL) para construir uma visualização de dados multi-dimensionais. Como teste usaremos o conjunto de dados iris, que contém 4 dimensões.
Crie uma página html chamada problema1.html. Esta página deve conter um combo box (drop-drown menu) que permite ao usuário escolher o parâmetro N que corresponde ao número de vizinhos mais próximos, utilizados para construir o grafo de vizinhança (como visto em sala). N deve variar de 5 a 15. Quando o valor do combo box muda de valor, o seu sistema deve montar o grafo de vizinhança correspondente e usar o FDL para mostrar o grafo na tela. Além disso, você deve colorir os pontos de acordo com o atributo species do dataset e incluir uma legenda para ser possível a identificação da espécie.

Problema 2

Neste problema usaremos Treemaps para visualizar despesas públicas no ano de 2015. Estamos interessados em investigar os seguintes orgãos: GABINETE DO PREFEITO, GABINETE DO VICE-PREFEITO, SECRETARIA DE EDUCAÇÃO, SECRETARIA DE MOBILIDADE E CONTROLE URBANO, SECRETARIA DE SAÚDE, SECRETARIA DE MULHER, SECRETARIA DE HABITAÇÃO, SECRETARIA DE ENFRENTAMENTO AO CRACK E OUTRAS DROGAS, SECRETARIA DE IMPRENSA e SECRETARIA DE JUVENTUDE E QUALIFICAÇÃO PROFISSIONAL. Note que no conjunto de dados contém gastos para estes órgãos e que cada órgão pode conter uma ou mais unidades.
Use o Treemap Layout do D3 para montar uma visualização das despesas de cada um dos órgãos acima. A hierárquia montada deve ser: Receife - Órgão - Unidade. O tamanho dos retângulos deve ser proporcional ao tamanho da despesa total feita pelo órgão.

Problema 3

Na aula 8 (sobre visualização de mapas), implementamos um choropleth dos estados unidos usando projeções em d3. Implemente interações de zoom e pan neste mapa. Além disso, implemente um tooltip para cada estado no mapa de tal forma que quando o mouse passar por cima de um estado, o nome do mesmo e os dados de produtividade (dados mostrados no choropleth) devem ser mostrados no tooltip.
