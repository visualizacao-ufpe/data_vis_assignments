Problema 1

O objetivo deste problema é estudar as ocorrências aeronáuticas que ocorreram na aviação civil brasileira. Para tanto, usaremos este dataset. Sua tarefa é implementar um sistema de visualização que permita mapear as ocorrências dadas no arquivo anterior usando as ferramentas de projeções geográficas do D3. Precisamos de três visualizações. A primeira, as ocorrências devem ser mostradas como pontos sobre o mapa de municípios do Brasil (dados). Os pontos devem ser coloridos de acordo com a coluna "ocorrencia_classificacao", vermelho para acidente, azul para incidente e verde para outros valores. A segunda visualização deve mostrar um choropleth do número de ocorrências por estado (dados). Escolha uma escala de cores apropriada e mostre uma legenda para as cores escolhidas. Finalmente, a tercerira visualização deve mostrar o número de ocorrências por município brasileiro. Novamente, escolha uma escala de cores apropriada e mostre uma legenda. O usuário deve ser capaz de mudar de uma visualização para outra por meio de um radio button. Use como referência o capítulo de Geomapping do livro de referência acima.

Problema 3

Neste problema estudaremos a taxa de óbitos por AIDS nas regiões metropolitanas do Brasil (% de óbitos por AIDS - Regiões Metropolitanas). O conjunto de dados contém dados para cada região metropolitana de 1990 até 2009. Ao invés de explorar estes dados por meio de visualizações de timeseries diretamente, usaremos a abordagem de redução de dimensionalidade para tentar buscar semelhanças entre regiões metropolitanas. Para isso, use o TSNE (veja exemplo de uso do código neste link) para projetar estes dados, assumindo que cada ano corresponde a uma dimensão dos dados. Uma vez que os dados estejam projetados, plote os mesmos por meio de círculos, um para cada região metropolitana. Deve ser possível ver o nome de cada região ao se passar o mouse por cima da mesma.
